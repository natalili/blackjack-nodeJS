Верстка:
-lobby
1. Кнопка создать стол
2. поле с перечнем столов которые:
- созданы
- в них еще нет 5 игроков
- не начата игра
3. при клике на стол открывается модальное окно с вопросом присоединиться к столу или отказаться
4. стол исчезает из доступных если:
- есть 5 игроков;
- начата игра
- есть хоть одна игра
5. при создании стола или при присоединении к нему lobby убирается table создаётся (желательно ajax или localStorage)
-table
6. На столе только для создателя доступна кнопка старт игры. У остальных такой нет.
7. После старта кнопка старта скрываться.
8. Кнопка выхода из игры
9. 5 мест для игроков:
10. аватар
11. полное имя
12. место под карты
13. сумма очков
14. ставка
15. Баланс текущего игрока
16. фишки для ставок
17. место для сообщений
18. кнопки для выбора действия
19. место для карт крупье
20. место для суммы очков крупье
21. место с таймером по раздумью
22. место для чата
23. при выходе всех игроков стол закрывается
24. в profile должен быть виден баланс
25. баланс можно пополнить на 1000
26. для нового пользователя первоначальный баланс 1500
27. при выходе из игры переход в lobby
Для игры:
28. при выходе игрока из игры и при разрыве соединения его баланс из table переноситься в user
При создании стол должен иметь:
29. перемешанную колоду ( хорошо если с возможностью на 3, 5... колод)
30. массив игроков, а у них:
31. id из user
32. hand для карт на руках
33. bet — текущая ставка
34. balans — при добавлении к столу списывается с user
35. avatar
36. fullName = fname + lname или Аноним если fname, lname - пусты
37. посчитанные очки
38. крупье:
39. hand — для карт
40. посчитанные очки
41. выдавать карту. т.е. из стола брать в игрока/крупье перекладывать
42. принимать ставку
43. при ставке с баланса списываться на ставку записывается
44. подсчет карт при получении
45. крупье берет до 16
46. туз 11, 10, 1 по удобству
47. пользователи ходят по порядку подключения
48. у каждого пользователя по одной ставке
49. если перебор, то не предлагать брать карту т.е. переход хода
50. если блекджек, то не предлагать брать карту т.е. переход хода;
51. игра против крупье т.е. очки при подсчете сравнивать с крупье;
52. выигрывает тот у кого больше, но не больше 21
53. при выигрыше ставка * 1,5
54. при ничьей ставка * 1
55. при проигрыше ставка * 0
56. после пересчета балансов новые ставки
57. тем кто перешел к столу не нужно получать список доступных столов
58. таймер на прием ставки. При окончании таймера выход из игры. Таймер 30 сек.
59. таймер на ход. При окончании таймера авто пас. Таймер 30 сек.
60. возможность при ходе выбрать ответ пас/взять карту.
